Alkebulan WebSocket Routes

The Alkebulan system provides real-time chat for document retrieval and processing.

WebSocket Endpoints:

1. Chat Route for Querying Winter Sports Data

  • Endpoint: ws://{baseUrl}/chat/similarity-search
  • Description: Queries winter sports data using Qdrant.
  • Usage: Send queries, receive real-time results.
  • Payload Format: Request: plain text query, Response: plain text response.

2. Chat Route for Querying Financial Data

  • Endpoint: ws://{baseUrl}/chat/open-search
  • Description: Queries financial data stored in OpenSearch.
  • Usage: Send financial queries, receive real-time results.
  • Payload Format: Request: plain text query, Response: plain text response.

3. Chat Route for Querying RAG Chain

  • Endpoint: ws://{baseUrl}/chat/rag-chain
  • Description: Queries a RAG chain for complex insights.
  • Usage: Send complex queries, receive comprehensive responses.
  • Payload Format: Request: plain text query, Response: plain text response.
